# ChatMistralAI

## Prerequisite

1. Register a [Mistral AI](https://mistral.ai/) account
2. Create an [API key](https://console.mistral.ai/user/api-keys/)

## Setup

1. **Chat Models** > drag **ChatMistralAI** node

2. **Connect Credential** > click **Create New**
