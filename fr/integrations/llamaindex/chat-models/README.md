# Modèles de chat

***

Les modèles de chat prennent une liste de messages en entrée et renvoient un message généré par le modèle en tant que sortie. Ces modèles tels que ** GPT-3.5-turbo ** ou ** GPT4 ** sont puissants et moins chers que ses modèles de complétion prédécesseur tels que ** text-davincii-003 **.
